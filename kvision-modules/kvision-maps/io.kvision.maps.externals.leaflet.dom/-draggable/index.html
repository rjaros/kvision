Draggable

open external class Draggable(element: HTMLElement, dragStartTarget: HTMLElement = definedExternally, preventOutline: Boolean = definedExternally) : Evented(source)

A class for making DOM elements draggable (including touch support). Used internally for map and marker dragging. Only works for elements that were positioned with DomUtil.setPosition.

See https://leafletjs.com/reference.html#draggable
