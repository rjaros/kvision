LayerGroup

open external class LayerGroup(layers: Array<Layer<*>> = definedExternally, options: Layer.LayerOptions = definedExternally) : Layer<Layer.LayerOptions> (source)

Used to group several layers and handle them as one.

If you add it to the map, any layers added or removed from the group will be added/removed on the map as well. Extends Layer.
